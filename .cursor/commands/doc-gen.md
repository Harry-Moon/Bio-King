# doc-gen

Génère une documentation technique complète pour le fichier ou le bloc de code sélectionné.
Inclus :

1. Une description JSDoc pour chaque fonction (paramètres, types, retour).
2. Un résumé au format Markdown expliquant l'objectif global du module.
3. Un exemple d'utilisation concret (code snippet).
   Réponds en français.

This command will be available in chat with /doc-gen
